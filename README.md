# My personal Neovim config

## Requirements
- [Neovim](https://github.com/neovim/neovim)
- [vim-plug](https://github.com/junegunn/vim-plug)
- [stylua](https://github.com/JohnnyMorganz/StyLua) (StyLua is the only lua formatter i found that works with lua 5.4 out of the box)

## Instructions
- Install vim-plug per instruction

- Download stylua prebuilt binary and place it in home folder

- Clone repo to folder ~/.config/nvim

```
git clone git@github.com:prometheusalpha/neovim-config.git ~/.config/nvim
```

- Run vim-plug command in neovim to install plugins

```
:PlugInstall
```
